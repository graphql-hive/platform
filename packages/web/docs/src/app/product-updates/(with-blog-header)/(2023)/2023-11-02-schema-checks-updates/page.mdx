We have added two new filters to the schema checks page:

- **Show only changed schemas** - select to view only schema checks with changes.
- **Show only failed checks** - select to view only failed schema checks.

![Schema checks filters](/changelog/2023-11-02-schema-checks-updates/screenshot-filters.png)

Visit the `Schema Checks` page to see it.
