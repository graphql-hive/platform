# Troubleshooting

This document contains some common troubleshooting tips for self-hosting Hive.

## Service logs and verbosity

All Hive services support setting the log level via the `LOG_LEVEL` environment variable. The
default log level is `INFO`. The following log levels are supported:

- `debug`
- `info`
- `warn`
- `error`
- `fatal`
- `silent`
